Photo Albums Page

During my first CO-OP, I used my free time to learn HTML and CSS because it was fun to make websites. My first project that I set for myself was to create a photo gallery and organize most photos on my phone. Using online resources, I developed my skills in the markup language. While working on the website, I also familiarized myself more with GitHub repositories and actions. I have a finished product where I will add new photos to new albums at the end of each month.
