Photo Gallery Website

Photo Albums Page

During a CO-OP term, I used my free time to learn HTML, CSS, and Javascript because I was looking for creative birthday gift. My first project that I set for myself was to create a photo gallery and organize most photos on my phone. Using online resources, I developed my skills in the markup language. While working on the website, I also familiarized myself more with GitHub repositories and actions. I have a finished product where I will add new photos and features to albums at the end of each month. Since it's completion I have made the page more responsive, added a "scroll-up" button using JS, and currently am working on creating a database for the photos.
