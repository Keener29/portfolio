Rogers Telecommunications

Backend Software Development

During my internship as a software developer at Rogers, I undertook three impactful projects that showcased my technical skills and contributed to the company's efficiency.

In Project 1, I harnessed the power of Voronoi diagrams to design an automation solution that significantly reduced manual labor. By utilizing Python libraries such as geopandas and geovoronoi, I developed a tool that efficiently demonstrates cell coverage across Canada. This project not only showcased my programming prowess but also exhibited the potential for automation to enhance complex geographical analyses.

Project 2 demonstrated my proficiency in integration and automation. Through the integration of Jira REST API and Rogers' proprietary backend API, I created a program that automates the publishing of "automation ideas" into a developer-friendly framework. This initiative streamlined communication and ideation processes within the development team, contributing to a more efficient workflow.

In Project 3, I tackled the challenge of data reporting and auditing. I developed an automated Excel spreadsheet auditing report that runs monthly checks for modem compliance abuse by Rogers employees. This solution showcased my attention to detail and commitment to maintaining data integrity within the company.

Throughout my internship, I demonstrated adaptability, technical acumen, and problem-solving skills. These projects collectively underscored my ability to leverage software solutions to drive automation, streamline processes, and contribute to the overall success of Rogers.
